Installation
============

Basic Installation
------------------

.. code-block:: bash

   pip install scitex

With Optional Dependencies
--------------------------

.. code-block:: bash

   # With GUI support
   pip install scitex[gui]

   # With web support
   pip install scitex[web]

   # All optional dependencies
   pip install scitex[all]

Development Installation
------------------------

.. code-block:: bash

   git clone https://github.com/ywatanabe1989/scitex.git
   cd scitex
   pip install -e ".[dev]"
